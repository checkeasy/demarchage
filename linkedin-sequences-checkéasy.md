## GUIDE D'UTILISATION MULTI-CANAL

### Principe général
- **Email** : Pour la substance (cas d'usage, ressources, propositions concrètes)
- **LinkedIn** : Pour la relation (curieusité, questions ouvertes, touchpoints)
- **Timing** : Jamais deux canaux le même jour

### Checklist avant envoi
- [ ] Nom et prénom correct
- [ ] Entreprise ou ville personnalisée
- [ ] Pas de lien dans demande de connexion
- [ ] Pas de salutation formelle ("Madame, Monsieur")
- [ ] Pas de "solution innovante" ou jargon
- [ ] Message court, lisible sur mobile
- [ ] Signature "Adrien" pour LinkedIn
- [ ] Pas plus de 1 emoji par message (optionnel)

### Règles de pause
- Après 3 tentatives sans réponse : attendre 15 jours
- Après 5 tentatives totales : archiver le prospect
- Si réponse négative : respecter et garder le contact (pas d'acharnement)

## NOTES IMPORTANTES

1. **Ton humain** : Ces messages ne doivent PAS sembler générés. Relis-les à haute voix.
2. **Pas de pitch** : Le but c'est une conversation, pas une vente.
3. **Timing** : Respecte les jours recommandés, mais adapte à ta capacité.
4. **Tracking** : Note les réponses et ajuste l'angle si besoin.
5. **Archivage** : Si 3 refus clairs ou 5 tentatives sans réponse → archiver.
